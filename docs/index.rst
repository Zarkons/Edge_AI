EDGE AI Documentation
=====================

This is the documentation for the Edge AI project. It provides an overview of the project, its components, and how to use it effectively.
This is a personal project and is not affiliated with any company or organization. It is started as my curiosity to learn more about Edge AI, and passion for sharing knowledge, with the hope that it will be useful to others, and potentially be formulated into some training materials in the future.

The main goal is to:
 - Provide a basic understanding of Edge AI concepts and technologies, focused on practical applications and real-world use cases.
 - Provide a documentation that is easy to follow and understand, that demonstrates that documentation plays a crucial role in the development, and gives an insight of how to structure and organize a project for better maintainability and scalability.
