Simple Audio Recognition
========================

This example demonstrates how to use the Light Edge AI framework to build a simple audio recognition application. The application is designed to recognize specific audio commands from input audio data, either from pre-recorded audio files or directly from a microphone.

The example includes training a model to recognize specific audio commands, quantizing the model for efficient execution on edge devices, and running inference either on a pre-recorded audio file or directly from microphone input,
using RT Lite Micro framework.
