Simple Audio Recognition
========================

This example demonstrates how to use the Edge AI framework to perform simple audio recognition tasks.

The example includes training a model to recognize specific audio commands, quantizing the model for efficient execution on edge devices, and running inference either on a pre-recorded audio file or directly from microphone input.
